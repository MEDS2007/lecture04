# HTML & CSS: **BOX MODEL** - **margin:** space *outside* - **padding:** space *inside* - **border:** line *around*
# **HTML** GOES IN **BOXES** Every HTML element is contained in a “box”: ![A diagram of the box model](images/boxmodel.png)
# THE **MARGIN** Makes an invisible **space around** an element usually set in **pixels**, **ems** or **percentages**: `1px`, `.5em`, `5%` Can be zero: `0` These elements have **default margins** (see lecture demo): `body h1..h6 p ul ol dl dd dir blockquote fieldset form`
# THE **PADDING** Is **inside** the box, **around** the box **content** **background colour**/**background image** extend into padding usually set in **pixels**, **ems** or **percentages**: `1px`, `.5em`, `5%` Can be zero: `0`
# THE **BORDER** Goes **around the outside** of the box usually set in **pixels**, sometimes **ems** or **percentages**: `1px`, `.5em`, `5%` Can be zero: `none`
# BOX-MODEL **SYNTAX**: 1 ## For both **margin** and **padding** ```css header { margin: 2px; /* a 2-pixel margin all round */ padding-top: 1em; /* one line of padding at top */ margin-left: 5%; /* 5% of available space to left */ } /* you can use: -top -right -bottom -left */ ```
# BOX-MODEL **SYNTAX**: 2 **CSS shorthand** - **4** values: **top, right, bottom, left** (clockface) - **3** values: **top, left-right, bottom** - **2** values: **top-bottom, left-right** ```css header { /* 4: top right bottom left: */ padding: 1em 10px .5em 5px; /* 3: one line top, 10 pixels l-r, 1/2 line bottom */ padding: 1em 10px .5em; /* 2: one line top+bottom, none left-right: */ margin: 1em 0; } ```
# BOX-MODEL **SYNTAX**: 3 Set top, right, bottom, left, then **reset one** ```css header { margin: 10px; /* 10 pixels all round… */ margin-top: 0; /* …but no top margin */ } ``` The second line comes *after*—so **overrides**—the first
# BOX-MODEL **SYNTAX**: 4 To **centre** an block element by itself, set the **left** and **right** margins to `auto`: ```css header { margin: 0 auto; /* none top-bottom, l-r auto */ } ``` `auto` fills the available space. This is a common way to **centre a page design** in the **browser window**, although current designs prefer to **fill the browser window** completely.
# BOX-MODEL **SYNTAX**: 5 By default, **padding and border are added** to the width and height of a box: ```css main { width: 70%; box-sizing: border-box; padding: 1em; border: 2px; } /* width is 70% + 2em + 4px */ ```
# BOX-MODEL **SYNTAX**: 6 To make measurements easier, set `box-sixing` to `border-box` to **include padding and border** in the width and height: ```css main { width: 70%; box-sizing: border-box; padding: 1em; border: 2px; } /* width is still 70% */ ```
# BOX-MODEL **SYNTAX**: 7 `display: inline-block` turns **inline elements** into **blocks that line up horizontally**, often for `a` or `li` tags in menus or the `img` tag for images. ```css a { display: inline-block; } /* 'block' will turn inline elements to blocks */ ``` [W3Schools: HTML Block and Inline Elements](http://www.w3schools.com/html/html_blocks.asp)
# BOX-MODEL **SYNTAX**: 8 - Learn about the box model with [exercises on W3Schools](http://www.w3schools.com/css/css_boxmodel.asp) - Adjust sliders on an [Interactive box-model demo](http://guyroutledge.github.io/box-model/)
# BOX-MODEL **BORDER**: 1 Draws a line around a box, has three parameters: ```css img { border-color: white; border-style: solid; /* dotted, dashed, (more)... */ border-width: 2px; } ```
# BOX-MODEL **BORDER**: 2 Each property can have **one**, **two**, **three** or the **four** “clockface” values: **top**, **right**, **bottom**, **left**. ```css header { border-color: white black; /* top, bottom */ border-width: 2px 0 4px; /* top, r-l, bottom */ border-style: none solid dotted dashed; } ```
# BOX-MODEL **BORDER**: 3 **CSS shorthand** for a single border style all around: ```css img { border: 1px #555 dotted; /* a grey dotted border */ } ```
# BOX-MODEL **BORDER**: 4 boxes don't have to look boxy. They can be a bit rounded… ```css img { border-radius: 10px; } ```
# BOX-MODEL **BORDER**: 5 Or have one, two or three sides rounded… ```css img { border-top-left-radius: 50%; border-bottom-left-radius: 50%; /* a semi-circle (if height=width) or half-ellipse */ } img { border-top-right-radius: 100%; /* an arc (if height=width) */ } ```
# BOX-MODEL **BORDER**: 5 Or completely round… ```css img { border-radius: 50%; } ```
